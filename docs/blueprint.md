# **App Name**: FinTrack Pro

## Core Features:

- Multiple Profiles: Allow users to create and manage multiple financial profiles within a single account.
- Income Tracking: Record income details: debts, insurance, client, source of income, amount, and date.
- Expense Tracking: Log expenses with date, amount, category, item/service, and frequency (daily, monthly, annually).
- Report Generation: Generate weekly, monthly, and annual reports for income and expenses.
- Profit/Loss Calculation: Calculate profit and loss per profile, including profit margin.
- Savings Management: Track savings accounts, investments, and amounts.
- Debt Management: Manage debt information including credit cards, house payments, car payments, mortgage, initial cash, new balance, and other debts.
- Transaction Logging: Record all transactions: amount, mode of payment, category of payment, recipient, and date.

## Style Guidelines:

- Primary color: Deep sky blue (#4169E1) for trust and stability.
- Background color: Light grayish-blue (#D1DCEB) for a clean, professional look.
- Accent color: Sky blue (#87CEEB) for highlighting key data and interactive elements.
- Body and headline font: 'Inter' sans-serif, providing a modern, objective, neutral look, suitable for both headlines and body text.
- Use minimalist icons for categories and actions, ensuring clarity and ease of use.
- Employ a clean and organized layout with clear sections for each feature to facilitate easy navigation.
- Use subtle transitions and animations to provide feedback and enhance user engagement.
